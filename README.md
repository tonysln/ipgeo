# IPGeo

Simple tools written in Python (in a Jupyter Notebook) using free APIs and packages, including:

🌎 IP Geolocation Visualizer

☁ Simple Weather Data

💻 WHOIS Lookup

🛰 Landsat Imagery

APIs used:

- [freegeoip.app](https://freegeoip.app/)
- [7timer](http://www.7timer.info/doc.php?lang=en#api)
- [NASA Earth](https://api.nasa.gov/)
